Border option will add... well a border to our element. It is set with 3 params:

  1. Number of pixels width
  2. Color
  3. Type of border style (solid for example)

We can use parameters as border-style,border-spacing, border-radius and so on. Even in format like in padding and margins with 4 values at once.
